# CHANGELOG

## [2.0.0] - 2026-06-11

### Changed
- **制限時間を10分から5分に短縮**。スポーンタイムライン・波・敵HPスケール・経験値曲線・ボスHPを5分用に再調整し、自動プレイシミュレーションで再検証（全51テスト通過）
- タイトル・OGP・共有カード・プロローグの「10分」表記をすべて「5分」に更新（OGP画像も再生成）

### Added
- **クリア時のエンディングシネマ**（映画調・自動進行・クリック/ESCでスキップ可）
  - レターボックス＋明朝体字幕＋オルゴール調エンディングテーマ（プロシージャル生成）
  - C1: 剣の墓標が並ぶ戦場に朝が来る（空の色が夜→暁にトランジション）
  - C2: 木漏れ日の森を家族のもとへ歩く背中
  - C3: 駆け寄る子どもたちを膝をついて抱きとめる再会
  - C4: 焚き火を囲む家族と蛍、「約束の肉」
  - 「ORC SURVIVOR — 完 —」のタイトルカード → リザルトへ
